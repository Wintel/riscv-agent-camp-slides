# 零基础 RISC-V 芯片智能体开发 · 夏令营

上海交通大学夏令营 **「零基础 RISC-V 芯片智能体开发」** 教学材料。第 1 课讲义：通过连接 RISC-V 板子（Milk-V Titan），学习 Linux 基础、网络与 SSH 远程连接，为后续"在板子上开发 AI 智能体"打底。参考 MIT [Missing Semester 中文版](https://missing-semester-cn.github.io/)。

纯 HTML 单文件幻灯片：**可离线、可投影、动画丰富、易编辑**，无需安装任何软件。

## ▶️ 如何放映

1. 双击 `linux-camp-slides.html`，用浏览器（Chrome / Edge 均可）打开。
2. 按 **`F`** 进入全屏放映。

| 操作 | 按键 |
|---|---|
| 下一步 / 下一页 | `→`、`空格`、`Enter`、鼠标左键点击 |
| 上一步 / 上一页 | `←`、`Backspace` |
| 第一页 / 最后一页 | `Home` / `End` |
| 全屏切换 | `F` |
| 手机 / 平板 | 左右滑动 |

舞台固定 1280×720，会自动等比缩放铺满任意屏幕，投影仪上清晰。

## ✏️ 如何编辑

无需任何工具，用记事本 / VS Code 直接改 html 即可。

- **加一页**：复制一整段 `<section class="slide" data-sec="章节名"> … </section>`。
- **让内容一个个蹦出来**：给元素加 `class="frag"`（想要"弹一下"再加 `pop`）。翻页时它们会按顺序显现，方便讲课节奏。
- **进场动画**：给元素加 `rise` / `rise-2` / `rise-3`，进入该页时自动淡入。
- **章节名**：改 `<section>` 上的 `data-sec="…"`，会显示在右上角。
- **换配色 / 字体**：只改文件顶部 `:root{ … }` 里的变量（见下方设计系统）。
- **改课程名**：改顶栏 `.brand` 里的文字。
- 页码、总页数由脚本自动计算，**不用手动改**。

### 终端窗口小抄
每一行都用 `<span class="ln">` 包起来，颜色类：
`u`=用户名(绿) · `p`=路径/提示符(蓝) · `cmd`=命令(白) · `out`=输出(灰) · `em`=强调(红) · `cm`=注释。
> 提示：正文里写重定向符号要用 `&gt;`（`>`）、`&lt;`（`<`）、`&amp;`（`&`），否则可能被浏览器当成标签。

## 🗂️ 第 1 课大纲（25 页）

- **开场（1–3）**：封面 · 今日目标 · 全景图（电脑→网络→Titan→Linux→AI）
- **Linux 基础（4–7）**：操作系统是什么 · Linux 无处不在 · 图形界面 vs 命令行 · 终端/Shell/提示符解剖
- **命令行（8–12）**：`pwd/ls/cd` · 文件系统树与路径 · 增删改查（含 `rm` 警告）· 求助三法（含"问大模型"）· 管道与重定向
- **网络（13–16）**：数据包怎么传 · IP 与局域网 · 找板子 IP + `ping` · 端口（22 号门）
- **SSH（17–21）**：什么是 SSH · 握手四步 · **★ 第一次连接 `ssh user@ip`** · 密钥登录 · `scp` 传文件
- **实战收尾（22–25）**：连上 Milk-V Titan（含板子规格）· 常见报错排查 · 下一课预告 + 回顾 · 谢谢 + 命令速查

## ⚠️ 使用前请替换
